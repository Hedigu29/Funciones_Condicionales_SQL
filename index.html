Funciones Condicionales en SQL

Lógica de negocio en tus consultas

El Rey CASE

Es el estándar universal (ANSI SQL).


SELECT producto,
  CASE 
    WHEN stock = 0 THEN 'Agotado'
    WHEN stock < 10 THEN 'Crítico'
    ELSE 'Disponible'
  END AS estado
FROM Inventario;

Tip: Siempre evalúa de arriba hacia abajo.

Manejo de Nulos

COALESCE: Devuelve el primer valor no nulo.


SELECT nombre, 
COALESCE(telefono, email, 'Sin contacto')
FROM Clientes;

Ideal para limpiar reportes con datos incompletos.

Atajos y Tips

En MySQL existe IF(cond, V, F):


SELECT IF(precio > 100, 'Caro', 'Barato') 
FROM Productos;
  • CASE: Potente y compatible.
  • COALESCE: Salvavidas para NULLs.
  • Alias: Usa siempre AS para claridad.
